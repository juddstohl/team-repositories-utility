# team-repositories-utility

Output a table listing your teams repositories and if they are frozen.

-  run npm install
-  Tested with node 16
-  Copy the .env.example -> .env
-  In the .env file, set the variables
-  node src/app.js

Output
| Index | RepoName | IsFrozen |
| ----------- | ----------- | --- |
| 0 | 'force-CSIDDP' | false |
